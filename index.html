swear.jank

discord swear counter + automation script

what is this

swear.jank is a simple automation idea that tracks how many times bad words are used in discord and optionally reacts to them.

features

  • tracks swear usage in chat logs
  • auto increments counter
  • faster than manual counting
  • hotkey control (start/stop)

stats concept

  • ~+1k swears/second (theoretical)
  • ~86M swears/day
  • ~31B swears/year

how to set it up

  1. download AutoHotkey v2 from official site
  2. create a file named swear.jank
  3. paste the script below
  4. save it (Ctrl + S)
  5. run it with AutoHotkey
